== Description ==

Easy-to-use tool to get rid of spam attacking your website AntiSpamLight is developed to help you to forget about annoying spam bots! No more spam and no more capcha! Just real information.

= Features =

- Protects site from spam bots.
- No need to bother with captcha.
- Provides strong protection of your site comments.
- Doesn't load your server.
- Easy to use.

**It's important**

Uses Javascript

== Installation ==

1. Upload `nix-antispam-light` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enjoy
